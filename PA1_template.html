There is 2304 missing values in the data set.

The strategy for filling in the missing values in the dataset is to impute them using the average number of steps taken per 5-minute interval across all days. A new variable steps2 is created that is equal to the original but with the missing data filled in.

steps2 = steps
for (i in 1:length(steps)){
     if (is.na(steps[i])){
          steps2[i]= steps.perint[row.names(steps.perint)==interval[i]]
     }
}
